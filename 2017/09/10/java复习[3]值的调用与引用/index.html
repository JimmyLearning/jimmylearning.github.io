java复习3| 值的调用与引用

简单的说，值调用是一种拷贝，引用调用是一种指向，引用调用的本质也是值调用。
一个方法可以修改引用传递所对应的变量值，而不能修改值传递所对应的变量值，这是按值调用与引用调用的根本区别。
理解值与引用调用可促进理解栈内存的概念。
